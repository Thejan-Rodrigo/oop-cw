Real-time Ticketing System

A platform for efficient ticket management and simulation.

Vendor Role

  • Single Ticket Addition: Manually add tickets to the pool.
  • Simulation: Initiate simulations with multiple vendors and customers.

Customer Role

  • Ticket Purchase: Purchase tickets from the pool.

The system utilizes an API polling mechanism to fetch real-time data from the backend, ensuring a dynamic and interactive user experience. The backend is designed to handle multiple requests and processes simultaneously, demonstrating its scalability and robustness.
